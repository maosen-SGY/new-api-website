三步开始使用（当前 API 流程）

1

查看文档

在 API 文档页确认目标接口、入参与返回字段结构。

2

购买接口商品

在定价页按“接口名 + 500 次”加入购物车并完成支付。

3

生成并复制密钥

前往个人中心生成一次性密钥，复制后完成系统接入。

快速示例（API）

curl -X POST https://www.kalodata.com/openapi/v1/tiktok/product/rank \
-H "secret-key: YOUR_API_KEY" \
-H "content-type: application/json;charset=UTF-8" \
-d '{"region":"US","language":"zh-CN","currency":"USD","date_range":"last7Day","page_number":1}'
